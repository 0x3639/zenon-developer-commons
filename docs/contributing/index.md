# Contributing

This section collects open questions and evaluation prompts for people who want to engage with the research — whether reviewing the architecture, implementing parts of it, or assessing it as a prospective architect.

- **[Architect Evaluation Questions](/contributing/architect-evaluation-questions)** — prompts for evaluating architectural understanding.
- **[ChangesHash Questions](/contributing/changeshash-questions)** — specific implementation questions on the ChangesHash model.

Contributions should stay technical, calm, and curiosity-driven. This is a research commons, not a governance channel.
